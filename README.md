# Descans Visual

Extensió web per a Firefox desenvolupada amb p5.js que transforma el temps de descans en una visualització abstracta basada en gotes, ones i un dipòsit d’aigua.

## Autor

Adrián Román Martin

## Descripció

Aquest projecte forma part del Repte 3. L’objectiu ha estat crear una extensió compatible amb Firefox utilitzant Manifest V3, una interfície amb elements DOM i persistència de dades mitjançant `storeItem()` i `getItem()`.

L’extensió funciona com un petit cronòmetre visual de descans:
- permet iniciar i pausar el temps,
- reiniciar el comptador,
- ajustar la intensitat visual,
- i conservar l’estat encara que es tanqui el popup.

## Funcionalitats principals

- Cronòmetre de descans
- Visualització abstracta del temps amb p5.js
- Botó d’inici/pausa
- Botó de reinici
- Slider d’intensitat visual
- Persistència de dades amb `storeItem()` i `getItem()`

## Tecnologies utilitzades

- HTML
- CSS
- JavaScript
- p5.js
- Web Extensions (Firefox)
- Manifest V3
